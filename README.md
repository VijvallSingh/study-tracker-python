# Study Tracker (Python)

## Overview
This is a simple command-line based study tracker built using Python.  
It allows users to record their study sessions, view summaries, and visualize their study time using graphs.

## Features
- Add study sessions with subject and time
- View total study time
- Subject-wise breakdown
- Graph visualization using matplotlib

## Technologies Used
- Python
- JSON (for data storage)
- Matplotlib (for graphs)

## How to Run

1. Make sure Python is installed
2. Install matplotlib:

pip install matplotlib

3. Run the program:

python main.py
